Démarrage

---------------------------------------------------------------------------------------

Préfixe : d$

Commande help : d$help

---------------------------------------------------------------------------------------

Commandes de bases

---------------------------------------------------------------------------------------

d$ping : Jouer au ping pong ;)

---------------------------------------------------------------------------------------

d$ServerInfos : Afficher les informations du serveur

---------------------------------------------------------------------------------------

d$welcome : Souhaiter la bienvenue à un membre

---------------------------------------------------------------------------------------

Modération

---------------------------------------------------------------------------------------

d$ban : Bannir un membre

Cette commande n'affecte pas les autres bots !

---------------------------------------------------------------------------------------

d$kick : Expulser un membre

Cette commande n'affecte pas les autres bots !

---------------------------------------------------------------------------------------

d$rename : Renommer un membre

---------------------------------------------------------------------------------------

d$role_add : Donner un rôle à un membre

---------------------------------------------------------------------------------------

d$role_remove : Enlever un rôle à un membre

---------------------------------------------------------------------------------------

Fun

---------------------------------------------------------------------------------------

d$cat : Envoyer une image de chat aléatoire

---------------------------------------------------------------------------------------

d$dog : Envoyer une image de chien aléatoire

---------------------------------------------------------------------------------------

d$gif : Envoyer un gif en rapport avec le mot entré

---------------------------------------------------------------------------------------

d$joke : Je vous fait une blague
